    Events
    An overall tournament can have up to seven different events being contested. For each of the seven, first indicate whether it is included in this tournament. Then, for each division, enter how many rounds there are (including semis and finals), and how many of those rounds are cumulative. Cumulative rounds start with the first round, and count the sum of a player's scores in all rounds up to that point. If no rounds are cumulative, the box can be left blank.
